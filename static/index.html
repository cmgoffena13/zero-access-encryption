Zero-access encryption (POC)

The server never receives your password or plaintext notes—only an SRP verifier and encrypted blobs. Unlock the vault in your browser with your password; PBKDF2 + AES-GCM uses your registration salt.

1. Register

2. SRP login

3. Vault (encrypted note)

Use the same password as registration. After register or SRP login, user id and salt are stored in this browser session for PBKDF2.
